다시점/경시적 자료(Repeated Measures / Longitudinal Data)란?

동일한 대상으로부터 시간 경과에 따라 반복적으로 측정되어 "개인 내 상관성"이 존재하는 데이터 구조임

  • 동일한 대상으로부터 시간의 흐름에 따라 반복적으로 측정된 데이터를 의미함.
  • 이러한 데이터는 개체 내 측정값들 사이에 상관관계가 존재한다는 중요한 특징을 가짐.
  • 임상 시험 및 의학 연구에서는 한 환자의 상태 변화를 추적하기 위해 여러 시점(예: Baseline, 1주차, 4주차)에 걸쳐 결과 지표를 반복 측정함.
  • 이로 인해 관측치 간의 독립성 가정이 깨지고 개인 내 상관성(Within-subject correlation)이 발생하므로 이를 고려한 특별한 통계 분석 방법이 필수적임.

방법비교 (분석 방법 특징 요약)

데이터의 특성과 연구 목적에 따라 아래 탭을 클릭하여 각 분석 기법의 심층이해 영역으로 이동해 보세요.

RM ANOVA

전통적인 반복측정 분산분석 기법임.

언제 사용해야 하는가?

결과 변수가 정규성을 충족하는 연속형 자료이며, 중간 탈락자로 인한 결측치 소실이 없는 완전한 데이터 구조에 사용함.

주요 특징

  • 그룹 효과, 시간 효과, 그리고 상호작용 효과 통합 검증
  • 결측치가 존재하는 경우 대상 자체를 완전 배제함
  • 가정 조건 요건(정규성, 등분산성, 구형성)이 엄격함

핵심 가정

  • 정규성 (Normality)
  • 등분산성 (Homogeneity of Variance)
  • 구형성 (Sphericity)

LMM (선형 혼합 모형)

RM ANOVA 의 구조적 한계를 보완 극복한 우수한 기법임.

언제 사용해야 하는가?

결과 변수가 연속형이며, 데이터 내 임의 결측이 섞여 있거나 개체별 고유 반응차(임의효과)를 규명해야 할 때 사용함.

주요 특징

  • 결측 수치 보유 피험자 정보를 폐기하지 않고 사용 가능
  • 환자 개인별 기초 레벨이나 변화율 등 고유 특이성을 모형화
  • 잔차 분포 공분산 구조를 다각도로 적용 설계함

핵심 개념

  • 고정 효과 (Fixed Effects)
  • 임의 효과 (Random Effects)

GEE (일반화 추정 방정식)

비정규 분포 범주형 종속변수에 유용한 모델임.

언제 사용해야 하는가?

결과 지표가 이분 범주형 및 빈도(Count) 자료형이며, 군집화된 개체 간 평균적 경향을 통계 규명해야 할 때 사용함.

주요 특징

  • 이항형, 포아송 등의 비정규 반응을 종단적으로 처리
  • 상관 구조 오선정 시에도 '샌드위치 견고 오차' 매커니즘 동원
  • 결측 요건으로 완전 임의 결측(MCAR) 상황을 전제함

핵심 개념

  • 집단 평균 추론 (Population-averaged)
  • 개인 고유의 임의 변동량 자체는 직접 추정 대상에서 제외

Repeated Measures ANOVA 통계 검정 조건

RM ANOVA 적용을 위한 세부 검정 논리 및 수학적 모델 식

RM ANOVA 모델 수식

$$Y_{ijk} = \mu + \alpha_i + \tau_j + (\alpha\tau)_{ij} + \pi_k + \epsilon_{ijk}$$
가정 1

정규성 (Normality)

각 관측 시점에 속하는 피험자들의 관측 오차가 정규분포를 명확히 따라야 함.

가정 2

등분산성 (Homogeneity of Variance)

각각의 독립 배정 그룹들이 균일하고 동등한 형태의 오차 분산 폭을 유지해야 함.

가정 3

구형성 (Sphericity)

임의의 서로 다른 두 시점 차이값 조합들의 분산 가중 크기가 고르게 평행 보존되어야 함.

Linear Mixed Model (선형 혼합 모형) 심층 이해

집단 일반화 평균(고정효과)과 환자별 특이선(임의효과)의 결합 원리

선형 혼합 모형(Linear Mixed Model, LMM)은 반복측정 데이터의 복잡한 구조를 효과적으로 다루는 강력하고 유연한 통계 모형임. 개체 간 이질성(개인차)과 개체 내 측정값 간의 상관성을 정교하게 동시에 모델링할 수 있다는 특장점이 존재함.

일반 선형 모형 (LM) 수식

$$Y_i = \beta_0 + \beta_1 X_{1i} + \dots + \beta_p X_{pi} + \epsilon_i$$

선형 혼합 모형 (LMM) 수식

$$\mathbf{Y}_k = \mathbf{X}_k \boldsymbol{\beta} + \mathbf{Z}_k \mathbf{b}_k + \boldsymbol{\epsilon}_k$$

고정 효과 (Fixed Effects)

연구자가 특정 수준(level)에 관심이 있고, 그 수준들이 모집단 전체를 대표한다고 가정하는 요인들임. 즉, 이 요인들의 효과는 특정 값으로 고정되어 있으며, 다른 연구에서도 동일하게 적용될 것으로 기대됨.

구체적 예시:
  • 성별: 남성/여성이라는 특정 범주
  • 치료 그룹: 신약 그룹/위약 그룹과 같이 연구자가 직접 설계한 그룹
  • 시간: 특정 시점(예: 0주차, 4주차, 8주차)의 효과
주요 특징:
  • 일반적으로 평균 차이나 회귀 계수로 해석됨.
  • 모집단 전체에 대한 일반화가 목적임.

임의 효과 (Random Effects, 랜덤 효과)

연구자가 특정 수준 자체보다는 그 수준들이 파생된 더 큰 모집단에 관심이 있고, 해당 수준들이 그 모집단에서 무작위로 추출되었다고 가정하는 요인들임. 이 요인들의 효과는 특정 값이 아니라 분포(예: 정규 분포)를 따른다고 가정함.

구체적 예시:
  • 환자 ID: 연구에 참여한 개별 환자들은 전체 환자 모집단에서 무작위로 추출된 것으로 간주됨.
  • 의료 기관: 여러 병원에서 데이터를 수집할 때, 각 병원은 전체 병원 모집단에서 무작위로 추출된 것으로 간주됨.
주요 유형:
  • 랜덤 절편 (Random Intercept): 개체마다 종속 변수의 시작점(평균)이 다름을 모델링함. (예: 환자마다 초기 BPRS 점수가 다름)
  • 랜덤 기울기 (Random Slope): 개체마다 독립 변수(예: 시간)에 따른 종속 변수의 변화율(기울기)이 다름을 모델링함. (예: 환자마다 시간에 따른 BPRS 점수 감소 속도가 다름)
특징:
  • 주로 분산 성분(Variance Component)으로 해석됨.
  • 개체 간의 이질성을 설명하고, 개체 내 측정값 간의 상관성을 간접적으로 설명함.

상관성 가정 (Correlation Structure) 심층 매트릭스

동일 대상 내 반복 측정치들 사이의 종속성 흐름을 사전 패턴화하는 요건임. LMM은 구형성 가정의 제약 없이 잔차 에러 공분산 구조를 임의 튜닝하여 고도로 정밀화된 분석을 수행함.

독립 (Independent)

$$\begin{pmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 1 \end{pmatrix}$$

복합 대칭 (CS)

$$\begin{pmatrix} 1 & \rho & \rho \\ \rho & 1 & \rho \\ \rho & \rho & 1 \end{pmatrix}$$

자기회귀 (AR(1))

$$\begin{pmatrix} 1 & \rho & \rho^2 \\ \rho & 1 & \rho \\ \rho^2 & \rho & 1 \end{pmatrix}$$

비구조화 (UN)

$$\begin{pmatrix} 1 & \rho_{12} & \rho_{13} \\ \rho_{12} & 1 & \rho_{23} \\ \rho_{13} & \rho_{23} & 1 \end{pmatrix}$$

GEE (일반화 추정 방정식) 심층 이해

비선형 링크 변환 구조를 결합한 모집단 평균 지향적 종단 예측 모델

일반화 추정 방정식(GEE, Generalized Estimating Equations)은 다양한 유형의 결과 변수에 적용 가능한 강력한 분석 방법임.

GEE 모델 수식 (링크 함수 활용)

$$g(E[Y_{ijk}]) = \beta_0 + \beta_1 X_{1ijk} + \dots + \beta_p X_{pijk}$$

언제 사용해야 하는가?

결과 변수가 범주형(예: 예/아니오)이거나 정규분포를 따르지 않을 때 주로 사용함.

주요 특징

  • 연속형, 범주형 변수 모두 분석 가능
  • 결측치가 있어도 분석 가능 (완전 임의 결측 가정 하에)
  • 개별 환자보다는 모집단 전체의 평균적인 변화에 초점을 맞춤

핵심 개념

  • 모집단 평균 추론 (Population-average): 전체 집단의 경향성을 파악하는 것이 주 목적임.
  • 랜덤 효과(Random Effects)는 직접 고려하지 않음: LMM과 달리 개인의 고유한 변화 궤적을 직접 모형화하지 않음.

주요 분석 패키지 설명

R 및 Python에서 경시적 자료 분석에 활용되는 핵심 라이브러리

R 패키지

afex

반복측정 분산분석(RM ANOVA)을 직관적으로 수행하기 위한 패키지. 구형성 가정이 위배되었을 때 자동으로 Greenhouse-Geisser 등의 보정된 p-value를 제공하여 실무에서 매우 편리함.

R 패키지

lme4 / nlme

선형 혼합 모형(LMM)의 핵심 패키지. lme4는 속도와 직관성이 좋고 복잡한 임의효과 처리에 능하며, nlme는 다양한 오차 상관구조(예: AR1)를 지정할 때 유용함.

R 패키지

geepack

일반화 추정 방정식(GEE)을 위한 범용 패키지. 로버스트 표준 오차 연산 기능이 우수하며, 범주형/카운트 데이터의 종단적 집단 평균 효과를 분석할 때 표준으로 사용됨.

Python 패키지

statsmodels

파이썬 통계 분석의 핵심. AnovaRM, mixedlm, gee 모듈을 통해 R과 동일한 수준의 반복측정 분석 및 혼합모형, GEE 구현을 지원함.

실무 코드 메뉴얼 (R / Python) 
# ==============================================================================
# [R실습] RM ANOVA 통계 가이드라인 (Wide 데이터 변환부터 사후대비 검정까지)
# ==============================================================================

# 1. 필요 패키지 적재
library(tidyr)
library(dplyr)
library(ggplot2)
library(car)
library(afex)
library(emmeans)
library(effectsize)

# 2. 재현 가능한 Wide-format 가상 데이터 빌드
set.seed(123)
data_wide <- data.frame(
  id    = 1:40,
  group = rep(c("A", "P"), each = 20),
  t0    = rnorm(40, mean = 50, sd = 10),
  t1    = rnorm(40, mean = 45, sd = 10),
  t2    = rnorm(40, mean = 40, sd = 10),
  t3    = rnorm(40, mean = 35, sd = 10)
)

# 3. Wide 구조를 정밀 분석용 Long 포맷 데이터 구조로 회귀 변환
long_data <- data_wide |>
  pivot_longer(cols = t0:t3, names_to = "time", values_to = "response") |>
  mutate(
    group = factor(group),
    id    = factor(id),
    time  = factor(time, levels = c("t0","t1","t2","t3"), ordered = TRUE) # 구형성 검정용 순서 팩터 지정
  )

# 4. 전통적인 일변량 aov() 기반 RM ANOVA 연산 실행
fit_aov <- aov(response ~ group * time + Error(id/time), data = long_data)
summary(fit_aov)

# 5. afex 기반 Mauchly 검정 + GG/HF 보정 p-value 검정 자동 도출
fit_afex <- aov_ez(
  id = "id", 
  dv = "response", 
  within = "time", 
  between = "group", 
  data = long_data, 
  type = 3
)
nice(fit_afex)

# 6. emmeans 기반 본페로니 보정 사후 단순 대비 검증
emm_res <- emmeans(fit_afex, ~ group | time)
pairs(emm_res, adjust = "bonferroni")
# [Python] statsmodels 패키지 활용 RM-ANOVA 검정

import numpy as np
import statsmodels.api as sm
from statsmodels.stats.anova import AnovaRM

# 가상 Long-format 데이터프레임 구조화 가정
# df_nrs columns: ['patient_id', 'group', 'time', 'nrs_score']

anova_model = AnovaRM(
    data=df_nrs,

# 아노바 스코어보드 결과 출력
print(res.summary())
TIP: 시점 요인은 R 포맷 변환 시 반드시 정형 순서 요인(`factor(ordered = TRUE)`)으로 선언해 주어야 구형성 검정이 명확해짐.

LMM 개별 궤적 선도

임의 효과 반영

치매 증상 악화 척도(CDR-SB)의 대표적 흐름(고정 효과선)과 환자별 기저 편차(점선)를 정밀화한 모델임.

수식적 임의 변화선

$$Y_{ij} = (\beta_0 + b_{0i}) + (\beta_1 + b_{1i})t + \epsilon_{ij}$$

환자 개개인마다 다른 출발 상태와 기울기 적응도를 모두 통계 행렬 안에 격리하여 오차 편향성을 최소화함.

실무 코드 메뉴얼 (R / Python) 
# ==============================================================================
# [R실습] LMM (선형 혼합 모형) 통계 가이드라인 (lme4 & nlme 패키지 비교)
# ==============================================================================

# 1. 필요 패키지 적재 및 데이터 정리
library(tidyr)
library(dplyr)
library(nlme)
library(lme4)
library(lmerTest) # lme4 적합 모델의 유의도(p-value) 환산용

# 2. 가상 시뮬레이션 데이터 구축 (Wide -> Long)
set.seed(123)
data_wide <- data.frame(
  id    = 1:40,
  group = rep(c("A","P"), each = 20),
  t0    = rnorm(40, mean = 50, sd = 10),
  t1    = rnorm(40, mean = 45, sd = 10),
  t2    = rnorm(40, mean = 40, sd = 10),
  t3    = rnorm(40, mean = 35, sd = 10)
)

long_data_lmm <- data_wide %>%
  pivot_longer(cols = starts_with("t"), names_to = "time_char", values_to = "response_value") %>%
  mutate(
    time    = as.numeric(sub("^t","", time_char)), # 공분산 AR(1) 연속형 연동용 시점
    time_f  = factor(time_char, levels = c("t0","t1","t2","t3"), ordered = TRUE), # 범주형 요인 시점
    id      = factor(id),
    group   = factor(group)
  )

# 3. [lme4] 임의 절편 및 임의 기울기를 수반한 고정교호 분석 모델 적합
# (time | id)는 환자 id 마다 각각의 고유 시작레벨과 속도 궤적을 임의효과로 지정함을 뜻함
fit_lmm <- lmer(response_value ~ group * time + (time | id), data = long_data_lmm)
summary(fit_lmm)

# 4. [nlme] 에러 잔차 간의 AR(1) 상관 구조 조건을 명시적으로 지정한 모델 적합
# corAR1(form = ~ time | id)를 통해 시점이 멀어질수록 지수 하락하는 오차 상관성 적용
fit_cat_ar1 <- lme(
  fixed       = response_value ~ group * time_f,
  random      = ~ 1 | id,                           # 환자별 임의 절편(출발선)만 지정
  correlation = corAR1(form = ~ time | id),         
  data        = long_data_lmm,
  method      = "REML"                              # 제한적최대우도 추정법 동원
)
summary(fit_cat_ar1)
# [Python] statsmodels 패키지 활용 LMM 적합
import statsmodels.api as sm
import statsmodels.formula.api as smf

# groups 파라미터에 피험자 고유 ID 선언
# re_formula="~time" 파라미터를 통해 시점별 임의 기울기 추가 주입
model_lmm = smf.mixedlm(
    "response_value ~ group * time", 
    data=df_long_lmm, 
    groups=df_long_lmm["id"], 
    re_formula="~time"
)
result = model_lmm.fit()

# 통계량 요약 출력
print(result.summary())
장점: 피험자의 추적 간격이 서로 다르게 엇갈린 불규칙한 데이터 상황에서도 뛰어난 복원 연산력을 제공함.

GEE 비율 축적 선도

범주형 결과변수

특정 치료 하에 환자들의 감염 증세 완치 비율(예/아니오)을 시점별 집단 비율로 추적한 바 차트임.

준우도 추정식의 강건함

오차의 종속적 연결 공식을 잘못 지정했을지라도 로버스트 표준오차를 수반해 통계 p-value 검정 가치를 완벽히 조율해 냄.

실무 코드 메뉴얼 (R / Python) 
# ==============================================================================
# [R실습] GEE (일반화 추정 방정식) 실무 적용 분석 전체 가이드라인
# ==============================================================================

# 1. 필요 패키지 적재 및 데이터 정의
library(tidyr)
library(dplyr)
library(geepack)

# 2. 이항 반응 변수(0/1)를 갖춘 가상 시뮬레이션 데이터 빌드
set.seed(123)
data_wide <- data.frame(
  id    = 1:40,
  group = rep(c("A","P"), each = 20),
  t0    = rnorm(40, mean = 50, sd = 10),
  t1    = rnorm(40, mean = 45, sd = 10),
  t2    = rnorm(40, mean = 40, sd = 10),
  t3    = rnorm(40, mean = 35, sd = 10)
)

long_data_gee <- data_wide %>%
  pivot_longer(cols = starts_with("t"), names_to = "time_char", values_to = "response_val") %>%
  mutate(
    time  = as.numeric(sub("^t","", time_char)),
    id    = factor(id),
    group = factor(group)
  ) %>%
  mutate(
    # 확률 예측치를 통한 이항 결과값(response_binary: 0 혹은 1) 산정
    response_binary = rbinom(n(), 1, prob = ifelse(group == "A", 0.3 + 0.05*time/3, 0.5 - 0.05*time/3)),
    time  = factor(time)
  )

# 3. geeglm() 함수를 적용한 GEE 범주 종속 모델 적합화 실행
model_gee <- geeglm(
  response_binary ~ group * time,
  data   = long_data_gee,
  id     = id,
  family = binomial(link = "logit"),
  corstr = "ar1"
)

# 4. Wald 검정 지표 기반 계수 및 통계량 요약표 산출
summary(model_gee)

# 5. 임상 유효성 판단을 위한 최종 오즈비(Odds Ratio) 추출
exp(coef(model_gee))
# [Python] statsmodels 활용 GEE 회귀 분석 적합
import statsmodels.api as sm
import statsmodels.formula.api as smf

# Binomial 분포 패밀리 및 AR(1) 임의 상관 구조 주입
fam = sm.families.Binomial()
cov_struct = sm.cov_struct.Autoregressive()

# "patient_id" 파라미터를 식별 그룹으로 묶어 연산
model_gee = smf.gee(
    "response_binary ~ group * time", 
    "patient_id", 
    df_long_gee, 
    cov_struct=cov_struct, 
    family=fam
)
result = model_gee.fit()

# 계수 출력 요약
print(result.summary())
강점: 결과변수가 포아송(Poisson), 다항(Multinomial) 범주일지라도 유연하게 링크 연산을 마침.

핵심 요약 및 선택 가이드

연구 목적과 데이터 형태에 따른 최적의 통계 기법 선택 플로우

1. 데이터가 완벽하고 연속형인가?

결측치가 전혀 없고, 종속변수가 정규분포를 따르는 연속형 데이터이며, 단순히 시간과 그룹 간의 교호작용(Interaction) 여부만을 검정하고 싶을 때.

RM ANOVA 선택

2. 결측치가 있거나 개인차가 중요한가?

종속변수가 연속형이나 중간 탈락자(결측치)가 존재하고, 피험자(개체)마다 고유한 기저 수준(Intercept)과 변화 속도(Slope) 등 랜덤 효과(Random Effect)를 명시적으로 모형화해야 할 때.

LMM 선택

3. 종속변수가 비정규형(범주/빈도)인가?

결과 변수가 이분형(예/아니오)이거나 포아송 범주형 데이터이며, 개별 특성보다는 모집단 전체의 평균적 경향(Population-Averaged)을 견고하게 추정하고 싶을 때.

GEE 선택
